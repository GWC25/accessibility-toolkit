# How to Use DigiWest Access Toolkit

## Getting Started
1. Open the Dashboard
2. Browse the Tool Finder
3. Click cards to flip and see solutions
4. Save helpful cards to My Toolkit
5. Track your progress with status badges

## Finding Tools
- Browse by category
- Click card to see solutions
- Watch video tutorials
- Follow step-by-step instructions

## My Toolkit
- To Explore (blue): Cards you've saved
- Developing (yellow): Tools you're learning
- Mastered (green): Tools you're confident with
- Click status badge to change
